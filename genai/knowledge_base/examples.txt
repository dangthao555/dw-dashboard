# OLIST SQL EXAMPLES

Question:
Doanh thu theo tháng

SQL:
SELECT
order_month_date,
ROUND(SUM(payment_value),2) AS revenue,
COUNT(DISTINCT order_id) AS total_orders
FROM gold_main_gold.fact_payments
WHERE order_status='delivered'
GROUP BY order_month_date
ORDER BY order_month_date;

Question:
Top 10 danh mục doanh thu cao nhất

SQL:
SELECT
dp.product_category,
COUNT(DISTINCT oi.order_id) AS total_orders,
ROUND(SUM(oi.price),2) AS revenue
FROM silver.order_items oi
JOIN gold_main_gold.dim_product dp
ON oi.product_id=dp.product_id
JOIN gold_main_gold.fact_orders fo
ON oi.order_id=fo.order_id
WHERE fo.order_status='delivered'
GROUP BY dp.product_category
ORDER BY revenue DESC
LIMIT 10;

---

Question:

SQL:

dc.customer_unique_id,
COUNT(DISTINCT fo.order_id) AS total_orders
FROM gold_main_gold.fact_orders fo
JOIN gold_main_gold.dim_customer dc

Question:
Điểm đánh giá trung bình theo trạng thái giao hàng

SQL:
SELECT
delivery_status,
ROUND(AVG(review_score),2) AS avg_review_score
FROM gold_main_gold.fact_orders
WHERE review_score IS NOT NULL
GROUP BY delivery_status
ORDER BY avg_review_score DESC;

---

Question:
Điểm đánh giá trung bình theo danh mục

SQL:
SELECT
dp.product_category,
ROUND(AVG(fo.review_score),2) AS avg_review_score
FROM gold_main_gold.fact_orders fo
JOIN silver.order_items oi
ON fo.order_id=oi.order_id
JOIN gold_main_gold.dim_product dp
ON oi.product_id=dp.product_id
WHERE fo.review_score IS NOT NULL
GROUP BY dp.product_category
ORDER BY avg_review_score DESC
LIMIT 10;

---
